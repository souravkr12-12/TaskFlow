# TaskFlow

A production-style Full Stack Task Management application built to learn and demonstrate modern backend and frontend development using FastAPI and React.

> 🚧 This project is currently under active development.

## 🚀 Getting Started

### Backend

```bash
cd backend

python -m venv .venv

# Windows
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn app.main:app --reload
```

Visit:

```
http://127.0.0.1:8000/docs
```
